# Changelog

All notable changes to the DSAgent VS Code extension will be documented in this file.

## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Sidebar chat panel with AI-powered data science assistant
- Sessions management tree view
- Variables inspector for kernel state
- Integration with DSAgent server via WebSocket
- Commands for analyzing Python code selections
- Commands for analyzing data files (CSV, Excel, JSON, Parquet)
- Status bar indicator showing connection state
- Human-in-the-loop (HITL) approval workflow
- Real-time plan visualization
- Code execution with output display
- Syntax highlighting for code blocks in chat
